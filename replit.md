# EcoMaker - Eco-Friendly Upcycling Platform

## Project Overview
EcoMaker is an innovative full-stack web application that transforms waste management into an engaging, community-driven experience. The platform provides comprehensive project guidance, material sourcing, and user-friendly sustainability tools to help users create meaningful upcycling projects.

## Core Technologies
- **Frontend**: React + TypeScript + Vite
- **Backend**: Express.js + TypeScript
- **Styling**: Tailwind CSS + shadcn/ui components
- **AI Integration**: OpenAI API for project generation and material analysis
- **External APIs**: Google Vision API, Materials Project API, Thingiverse API
- **Storage**: In-memory storage (MemStorage) for development
- **AR Features**: Three.js, A-Frame, MindAR for augmented reality visualization
